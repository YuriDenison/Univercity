F# Sample Readme - Web Crawl

Prerequisites

This sample requires F# which is included in Visual Studio 2010 Beta 2 or which is available as a downloadable CTP add-in for Visual Studio 2008. For more information and downloads please visit FSharp.net.

Description

This sample contains a single script containing all the code for retrieving HTML from URL's, parsing out the links, and recursively navigating to the discovered links.

Files

  • script.fsx - A commented script file that guides the user through making and parsing web requests. The code can be executed line by line in F# Interactive as you read the comments.
