F# Sample Readme - Tutorial

Prerequisites

This sample requires F# which is included in Visual Studio 2010 RC or which is available as a downloadable CTP add-in for Visual Studio 2008. For more information and downloads please visit FSharp.net.

Description

This program gives simple examples of how most of the language features work. Try highlighting the lines as you go and press Alt+Enter to execute the lines in F# Interactive. Equivalently, you can right-click on a line or a selection of text and choose Send to Interactive.

Files

  • Tutorial.fs - A commented code file guiding the user through a brief tour of some of the basic functionality found in F#.
