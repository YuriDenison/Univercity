F# Sample Readme - Word Automation

Prerequisites

This sample requires F# which is included in Visual Studio 2010 RC or which is available as a downloadable CTP add-in for Visual Studio 2008. For more information and downloads please visit FSharp.net. Word 2010, 2007, or 2003 must be installed as well.

Description

Demonstrates how to automate basic text manipulation, formatting, and spelling correction in Word.

Files

  • main.fs - A commented source file demonstrating interop with Word.
