F# Sample Readme - Excel Automation

Prerequisites

This sample requires F# which is included in Visual Studio 2010 RC or which is available as a downloadable CTP add-in for Visual Studio 2008. For more information and downloads please visit FSharp.net. Excel 2010, 2007, or 2003 must be installed as well.

Description

This program demonstrates basic data entry, retrieval, and charting in Excel.

Files

  • main.fs - A commented source file demonstrating interop with Excel.
