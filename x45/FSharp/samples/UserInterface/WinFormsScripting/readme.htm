F# Sample Readme - WinForms

Prerequisites

This sample requires F# which is included in Visual Studio 2010 Beta 2 or which is available as a downloadable CTP add-in for Visual Studio 2008. For more information and downloads please visit FSharp.net.

Description

This sample contains two scripts showing basic GUI creation for Windows Forms. The examples demonstrate working with menu controls, rich text boxes, and basic event handling.

Files

  • editor.fsx - A commented script file that demonstrates how to set up a basic text editor built around a RichTextBox control and menu items that create dialog boxes. The code can be executed line by line in F# Interactive.
  • helloform.fsx - Demonstrates creating a very basic UI with a text box displaying "Hello World".
