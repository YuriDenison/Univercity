F# Sample Readme - Solar System

Prerequisites

This sample requires F# which is included in Visual Studio 2010 RC or which is available as a downloadable CTP add-in for Visual Studio 2008. For more information and downloads please visit FSharp.net.

Description

This sample simulates the orbits of planets around the sun. The calculations use real data and the units of measure feature in F# allows the compiler to help verify that they are correct.

Files

  • SolarSystem.fs - Source file containing planet definitions, simulation logic, and graphics painting.
